# 死了都要 try

[用 JavaScript 写的“死了都要 try”](https://s7prods.github.io/try-until-die/)

## 🎵 项目来源

- **原曲**: 《死了都要爱》
- **创意来源**: [视频 BV1CDJEzfEQ7](https://www.bilibili.com/video/BV1CDJEzfEQ7)
  - 标题: 《程序员神曲 —— 死了都要try》
  - UP主: Victor_Yeh
  - 演唱者: 人工智能（AI）

## 💻 技术特色

- 使用 JavaScript 对象属性、方法调用输出歌词
- 通过 try-catch 语句实现歌词逻辑

## 🚀 运行方式

```bash
node .
```

或者直接在浏览器打开 index.html
